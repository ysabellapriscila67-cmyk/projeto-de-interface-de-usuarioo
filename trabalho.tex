# 🚀 NEBULA X

## Sobre o Projeto

O NEBULA X é uma interface gráfica desenvolvida em Python utilizando a biblioteca Tkinter. O projeto foi criado com uma proposta futurista e tecnológica, inspirada no universo espacial e em interfaces modernas. O nome "NEBULA X" foi escolhido por remeter às nebulosas do espaço, transmitindo inovação, mistério e modernidade.

O objetivo foi transformar uma interface simples em uma experiência mais impactante, moderna e visualmente atrativa para o usuário.

---

# 🎨 Paleta de Cores

As cores utilizadas foram escolhidas para criar uma estética futurista e cyberpunk.

- Fundo Principal: #0a0a0a
- Fundo dos Inputs: #1a1a1a
- Rosa Neon (Destaque Principal): #ff0055
- Roxo Neon (Destaque Secundário): #7b00ff
- Texto Principal: #f5f5f5
- Verde de Sucesso: #00ff88
- Vermelho de Erro: #ff1744

Essas cores foram utilizadas para criar um contraste forte e transmitir uma sensação de tecnologia, inovação e modernidade.

---

# 🏗 Estrutura da Interface

## Cabeçalho

No cabeçalho foram adicionados:

- Nome da marca: NEBULA X
- Slogan: Beauty Beyond Reality

Essa área foi criada para apresentar a identidade visual da marca logo no primeiro contato do usuário com o sistema.

---

## Área Principal

Na área principal foram adicionados:

- Campo para Nome
- Campo para E-mail
- Texto de apresentação
- Botão de acesso

Todos os elementos foram centralizados para criar uma interface limpa, organizada e moderna.

---

## Rodapé

Foi adicionado um rodapé para reforçar a identidade visual da marca e finalizar a interface de forma elegante.

---

# ⚡ Interatividade

Foi implementado um efeito Hover no botão principal.

Quando o usuário passa o mouse sobre o botão:

- A cor do botão é alterada.
- O elemento ganha mais destaque visual.

Esse recurso deixa a interface mais dinâmica e melhora a experiência do usuário.

---

# ✅ Validação dos Campos

Foi implementada uma validação para verificar se os campos foram preenchidos corretamente.

## Mensagem de Erro

Quando os campos estiverem vazios, o sistema exibe:

Ops!

Por favor, preencha todos os campos.

## Mensagem de Sucesso

Quando os dados forem preenchidos corretamente, o sistema exibe:

Bem-vindo(a) à NEBULA X!

Obrigado por se juntar a nós.

Em breve você receberá novidades no e-mail informado.

---

# 🎯 Objetivo do Projeto

O principal objetivo do projeto foi criar uma interface moderna e futurista que transmitisse:

- Tecnologia
- Inovação
- Exclusividade
- Modernidade
- Impacto Visual

Através da combinação das cores neon, do tema escuro, das fontes marcantes e dos elementos interativos, foi possível criar uma experiência visual mais envolvente e agradável para o usuário.
